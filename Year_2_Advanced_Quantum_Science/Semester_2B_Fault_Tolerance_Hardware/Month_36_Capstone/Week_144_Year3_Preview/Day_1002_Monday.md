### 1. The Transition: Coursework to Research

Year 3 represents a fundamental shift in your educational journey. While Years 0-2 focused on *absorbing* established knowledge through structured daily lessons, Year 3 emphasizes *synthesizing* that knowledge and preparing to *create* new contributions.

**The Three Phases of PhD Education:**

```
Phase 1 (Years 0-2): Knowledge Acquisition
├── Structured curriculum
├── Daily lessons and problems
├── Building foundational expertise
└── Output: Comprehensive knowledge base

Phase 2 (Year 3): Knowledge Integration
├── Qualifying examination preparation
├── Comprehensive review and synthesis
├── Research skill development
└── Output: Demonstrated mastery, research readiness

Phase 3 (Years 4-5): Knowledge Creation
├── Original research
├── Literature contribution
├── Thesis development
└── Output: Novel contributions to the field
```

### 4. Qualifying Exam Deep Dive

The qualifying examination is the gateway to PhD candidacy. Understanding its structure is essential for effective preparation.

#### Written Examination Format

$$\text{Written Exam} = 4 \text{ hours}, \text{ closed book}$$

**Section A: Quantum Mechanics (25%)**
- 3 problems, ~20 minutes each
- Topics: Postulates, time evolution, angular momentum, perturbation theory, identical particles

**Section B: Quantum Information (25%)**
- 3 problems, ~20 minutes each
- Topics: Density matrices, entanglement, channels, teleportation, Bell inequalities

**Section C: Quantum Error Correction (25%)**
- 3 problems, ~20 minutes each
- Topics: Stabilizer codes, surface codes, fault tolerance, thresholds

**Section D: Algorithms & Applications (25%)**
- 3 problems, ~20 minutes each
- Topics: Circuit model, Shor/Grover, variational methods, complexity

**Scoring:**
- Each problem: 0-10 points
- Section total: 30 points
- Passing: 70% overall with no section below 50%

#### Oral Examination Format

$$\text{Oral Exam} = 2 \text{ hours}, \text{ 3-4 committee members}$$

**Structure:**
1. **Technical Presentation** (30 min)
   - Present your research proposal
   - Demonstrate deep understanding of chosen area

2. **Technical Questions** (60 min)
   - Committee questions spanning all core areas
   - Probing depth of understanding
   - Connections between topics

3. **Research Discussion** (30 min)
   - Defend your research proposal
   - Demonstrate research readiness
   - Discuss potential challenges and approaches

class KnowledgeAssessment:
    """Track and visualize knowledge readiness for qualifying exam."""

    def __init__(self):
        # Define exam topics with weights
        self.topics = {
            'QM': {
                'weight': 0.25,
                'subtopics': [
                    'Postulates and formalism',
                    'Time evolution',
                    'Angular momentum',
                    'Perturbation theory',
                    'Scattering theory',
                    'Identical particles'
                ]
            },
            'QI': {
                'weight': 0.25,
                'subtopics': [
                    'Density matrices',
                    'Entanglement measures',
                    'Quantum channels',
                    'Protocols (teleportation, etc.)',
                    'Bell inequalities',
                    'Capacity theorems'
                ]
            },
            'QEC': {
                'weight': 0.25,
                'subtopics': [
                    'Stabilizer formalism',
                    'CSS codes',
                    'Surface codes',
                    'Fault tolerance',
                    'Threshold theorems',
                    'Decoding algorithms'
                ]
            },
            'Algorithms': {
                'weight': 0.25,
                'subtopics': [
                    'Circuit model',
                    "Shor's algorithm",
                    "Grover's algorithm",
                    'VQE/QAOA',
                    'Complexity classes',
                    'Query complexity'
                ]
            }
        }

        # Initialize scores
        self.scores = {
            section: {topic: 0 for topic in data['subtopics']}
            for section, data in self.topics.items()
        }

    def update_score(self, section, subtopic, score):
        """Update knowledge score (1-5 scale)."""
        if section in self.scores and subtopic in self.scores[section]:
            self.scores[section][subtopic] = score

    def get_section_average(self, section):
        """Calculate average score for a section."""
        scores = list(self.scores[section].values())
        return np.mean(scores) if scores else 0

    def get_overall_readiness(self):
        """Calculate overall readiness percentage."""
        total = 0
        for section, data in self.topics.items():
            section_avg = self.get_section_average(section)
            total += section_avg * data['weight']
        return (total / 5) * 100  # Convert to percentage

    def identify_weaknesses(self, threshold=3):
        """Find topics below threshold."""
        weak = []
        for section, subtopics in self.scores.items():
            for topic, score in subtopics.items():
                if score < threshold:
                    weak.append((section, topic, score))
        return sorted(weak, key=lambda x: x[2])

    def visualize_readiness(self):
        """Create radar chart of readiness by section."""
        sections = list(self.topics.keys())
        averages = [self.get_section_average(s) for s in sections]

        # Create radar chart
        angles = np.linspace(0, 2 * np.pi, len(sections), endpoint=False)
        averages_plot = averages + [averages[0]]  # Close the polygon
        angles_plot = np.concatenate([angles, [angles[0]]])

        fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

        # Radar chart
        ax1 = plt.subplot(121, polar=True)
        ax1.plot(angles_plot, averages_plot, 'b-', linewidth=2)
        ax1.fill(angles_plot, averages_plot, 'b', alpha=0.25)
        ax1.set_xticks(angles)
        ax1.set_xticklabels(sections)
        ax1.set_ylim(0, 5)
        ax1.set_title('Qualifying Exam Readiness by Section', fontsize=14, pad=20)

        # Add reference circles
        for r in [1, 2, 3, 4, 5]:
            ax1.plot(angles_plot, [r]*len(angles_plot), 'gray',
                    linestyle='--', alpha=0.3, linewidth=0.5)

        # Bar chart of subtopic scores
        ax2 = plt.subplot(122)
        all_topics = []
        all_scores = []
        colors = []
        color_map = {'QM': 'steelblue', 'QI': 'coral',
                    'QEC': 'seagreen', 'Algorithms': 'mediumpurple'}

        for section in sections:
            for topic in self.topics[section]['subtopics']:
                all_topics.append(f"{section}: {topic[:15]}...")
                all_scores.append(self.scores[section][topic])
                colors.append(color_map[section])

        y_pos = np.arange(len(all_topics))
        ax2.barh(y_pos, all_scores, color=colors, alpha=0.7)
        ax2.set_yticks(y_pos)
        ax2.set_yticklabels(all_topics, fontsize=8)
        ax2.set_xlim(0, 5)
        ax2.axvline(3, color='red', linestyle='--', alpha=0.5, label='Passing threshold')
        ax2.set_xlabel('Score (1-5)')
        ax2.set_title('Detailed Topic Readiness', fontsize=14)
        ax2.legend()

        plt.tight_layout()
        plt.savefig('year3_readiness_assessment.png', dpi=150, bbox_inches='tight')
        plt.show()

        return fig

class Year3StudyPlan:
    """Generate and track Year 3 study plan."""

    def __init__(self, start_date="2026-02-15"):
        self.start_date = datetime.strptime(start_date, "%Y-%m-%d")

        # Year 3 structure
        self.quarters = {
            'Q1': {
                'months': [37, 38, 39],
                'focus': 'Quantum Mechanics',
                'weeks': list(range(145, 157))
            },
            'Q2': {
                'months': [40, 41, 42],
                'focus': 'Quantum Information',
                'weeks': list(range(157, 169))
            },
            'Q3': {
                'months': [43, 44, 45],
                'focus': 'QEC & Hardware',
                'weeks': list(range(169, 181))
            },
            'Q4': {
                'months': [46, 47, 48],
                'focus': 'Comprehensive Prep',
                'weeks': list(range(181, 193))
            }
        }

        # Weekly hour targets
        self.weekly_hours = {
            'review': 10,
            'problems': 12,
            'practice_exams': 5,
            'research_reading': 4,
            'presentation': 2
        }

    def generate_timeline(self):
        """Generate timeline with key dates."""
        timeline = []
        current_date = self.start_date

        for quarter, data in self.quarters.items():
            for month in data['months']:
                month_start = current_date
                month_end = current_date + timedelta(days=28)

                timeline.append({
                    'quarter': quarter,
                    'month': month,
                    'focus': data['focus'],
                    'start': month_start.strftime("%Y-%m-%d"),
                    'end': month_end.strftime("%Y-%m-%d")
                })

                current_date = month_end + timedelta(days=1)

        return timeline

    def calculate_hours_needed(self, assessment):
        """Estimate hours needed based on assessment."""
        hours_per_section = {}

        for section in assessment.topics:
            avg = assessment.get_section_average(section)
            # More hours needed for lower scores
            base_hours = 100  # Base hours per section
            multiplier = (5 - avg) / 2  # 0-2x multiplier based on weakness
            hours_per_section[section] = int(base_hours * (1 + multiplier))

        return hours_per_section

    def visualize_timeline(self):
        """Create Gantt-style timeline visualization."""
        fig, ax = plt.subplots(figsize=(14, 6))

        colors = {'Q1': 'steelblue', 'Q2': 'coral',
                 'Q3': 'seagreen', 'Q4': 'mediumpurple'}

        y_positions = {'Q1': 3, 'Q2': 2, 'Q3': 1, 'Q4': 0}

        for quarter, data in self.quarters.items():
            start_week = data['weeks'][0]
            end_week = data['weeks'][-1]
            width = end_week - start_week + 1

            ax.barh(y_positions[quarter], width, left=start_week,
                   color=colors[quarter], alpha=0.7, edgecolor='black')
            ax.text(start_week + width/2, y_positions[quarter],
                   f"{quarter}: {data['focus']}",
                   ha='center', va='center', fontsize=11, fontweight='bold')

        ax.set_yticks(list(y_positions.values()))
        ax.set_yticklabels(list(y_positions.keys()))
        ax.set_xlabel('Week Number', fontsize=12)
        ax.set_title('Year 3 Qualifying Exam Preparation Timeline', fontsize=14)
        ax.set_xlim(144, 193)

        # Add month markers
        for month in range(37, 49):
            week = 145 + (month - 37) * 4
            ax.axvline(week, color='gray', linestyle=':', alpha=0.5)
            ax.text(week, 3.5, f'M{month}', ha='center', fontsize=9)

        # Mark exam
        ax.axvline(192, color='red', linestyle='-', linewidth=2)
        ax.text(192, -0.5, 'QUALIFYING\nEXAM', ha='center', fontsize=10,
               color='red', fontweight='bold')

        plt.tight_layout()
        plt.savefig('year3_timeline.png', dpi=150, bbox_inches='tight')
        plt.show()

        return fig

## Key Topics Covered

### Quantum Error Correction (Semester 2A)
- Classical error correction review
- Quantum error models (bit-flip, phase-flip, depolarizing)
- Three-qubit codes and Knill-Laflamme conditions
- Shor code (9-qubit) and Steane code (7-qubit)
- CSS code construction
- Stabilizer formalism and Pauli group
- Gottesman-Knill theorem
- Color codes and Reed-Muller codes
- Kitaev's toric code
- Surface codes and lattice surgery
- MWPM decoding

### Fault Tolerance & Hardware (Semester 2B)
- Fault-tolerant definitions and threshold theorem
- Eastin-Knill theorem and transversal gates
- Magic state distillation
- Universal fault-tolerant computation
- Superconducting qubits (transmon, flux)
- Trapped-ion quantum computing
- Neutral atom arrays and Rydberg gates
- Photonic quantum computing
- Silicon spin qubits
- HHL algorithm and quantum simulation
- QLDPC codes and constant-overhead QEC

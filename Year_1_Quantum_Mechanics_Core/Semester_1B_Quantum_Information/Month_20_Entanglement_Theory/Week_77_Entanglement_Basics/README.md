## Key Concepts

### Separability Definition
$$\rho_{sep} = \sum_i p_i \rho_i^A \otimes \rho_i^B, \quad p_i \geq 0, \sum_i p_i = 1$$

### Bell States
$$|\Phi^{\pm}\rangle = \frac{1}{\sqrt{2}}(|00\rangle \pm |11\rangle)$$
$$|\Psi^{\pm}\rangle = \frac{1}{\sqrt{2}}(|01\rangle \pm |10\rangle)$$

### Entanglement Witness
$$W: \text{Tr}(W\rho_{sep}) \geq 0 \text{ for all separable } \rho, \text{ but } \text{Tr}(W\rho_{ent}) < 0$$

### PPT Criterion
$$\rho \text{ separable} \Rightarrow \rho^{T_B} \geq 0$$

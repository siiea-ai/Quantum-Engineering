## The Quantum Foundation

### Mathematical → Quantum Dictionary

Every topic in Year 0 directly enables quantum mechanics:

| Year 0 Topic | Quantum Mechanics Application |
|--------------|------------------------------|
| Calculus | Wave functions, Schrödinger equation |
| Linear Algebra | Hilbert space, observables, measurements |
| Differential Equations | Time evolution, eigenvalue problems |
| Complex Analysis | Propagators, path integrals, scattering |
| Classical Mechanics | Hamiltonian → Schrödinger, {·,·} → [·,·]/iℏ |
| Functional Analysis | Rigorous quantum formalism, spectral theorem |
| Group Theory | Symmetries, angular momentum, selection rules |

### The Classical → Quantum Bridge

```
Classical Mechanics          Quantum Mechanics
─────────────────           ─────────────────
Observable F(q,p)    →      Operator F̂
Poisson {F,G}        →      Commutator [F̂,Ĝ]/iℏ
Phase Space (q,p)    →      Hilbert Space |ψ⟩
Hamilton H(q,p)      →      Hamiltonian Ĥ
{q,p} = 1            →      [q̂,p̂] = iℏ
Action S             →      Path Integral ∫e^{iS/ℏ}
Liouville ρ          →      Density Matrix ρ̂
```

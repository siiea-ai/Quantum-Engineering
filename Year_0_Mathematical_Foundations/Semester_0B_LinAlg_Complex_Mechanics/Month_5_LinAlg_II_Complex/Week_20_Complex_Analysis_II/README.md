## ðŸ”‘ Key Theorems

### Laurent Series
$$f(z) = \sum_{n=-\infty}^{\infty} a_n(z-z_0)^n$$

### Residue Theorem
$$\oint_C f(z)\,dz = 2\pi i \sum_{\text{inside}} \text{Res}[f, z_k]$$

### Argument Principle
$$\frac{1}{2\pi i}\oint_C \frac{f'(z)}{f(z)}\,dz = Z - P$$

### RouchÃ©'s Theorem
If |g| < |f| on C, then f and f+g have the same number of zeros inside C.

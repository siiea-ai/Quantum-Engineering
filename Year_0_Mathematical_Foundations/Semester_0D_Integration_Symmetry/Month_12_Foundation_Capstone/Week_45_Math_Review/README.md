## Key Formula Review

### Calculus
- Gradient: $\nabla f = \sum_i \frac{\partial f}{\partial x_i}\hat{e}_i$
- Divergence: $\nabla \cdot \mathbf{F} = \sum_i \frac{\partial F_i}{\partial x_i}$
- Stokes' theorem: $\oint_{\partial S} \mathbf{F} \cdot d\mathbf{l} = \int_S (\nabla \times \mathbf{F}) \cdot d\mathbf{A}$

### Linear Algebra
- Eigenvalue equation: $A\mathbf{v} = \lambda\mathbf{v}$
- Spectral decomposition: $A = \sum_i \lambda_i |\lambda_i\rangle\langle\lambda_i|$
- Completeness: $\sum_i |\lambda_i\rangle\langle\lambda_i| = I$

### Differential Equations
- Sturm-Liouville: $\frac{d}{dx}[p(x)\frac{dy}{dx}] + [q(x) + \lambda w(x)]y = 0$
- Green's function: $LG(x,x') = \delta(x-x')$
- Separation of variables: $u(x,t) = X(x)T(t)$

### Complex Analysis
- Cauchy integral: $f(z_0) = \frac{1}{2\pi i}\oint_C \frac{f(z)}{z-z_0}dz$
- Residue theorem: $\oint_C f(z)dz = 2\pi i\sum_k \text{Res}(f, z_k)$
- Laurent series: $f(z) = \sum_{n=-\infty}^{\infty} a_n(z-z_0)^n$

### Functional Analysis
- Inner product: $\langle f|g\rangle = \int f^*(x)g(x)dx$
- Operator adjoint: $\langle f|Ag\rangle = \langle A^\dagger f|g\rangle$
- Spectral theorem: Self-adjoint operators have real eigenvalues

### Group Theory
- Commutation: $[J_i, J_j] = i\hbar\epsilon_{ijk}J_k$
- Clebsch-Gordan: $|j_1j_2;jm\rangle = \sum_{m_1m_2} C^{jm}_{j_1m_1;j_2m_2}|j_1m_1;j_2m_2\rangle$
- Wigner-Eckart: $\langle j'm'|T^{(k)}_q|jm\rangle = \langle j'\|T^{(k)}\|j\rangle C^{j'm'}_{jm;kq}$

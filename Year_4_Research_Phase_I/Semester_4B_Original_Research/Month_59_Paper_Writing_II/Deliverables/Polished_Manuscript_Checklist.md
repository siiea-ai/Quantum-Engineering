## Part II: Structure Verification

### Title

- [ ] Concise (under 15 words unless necessary)
- [ ] Accurately describes content
- [ ] Contains key searchable terms
- [ ] No unnecessary jargon or acronyms
- [ ] Avoids sensationalism
- [ ] Meets journal character limit: ___/___

### Author Information

- [ ] All authors listed in agreed order
- [ ] All affiliations accurate and current
- [ ] Corresponding author designated
- [ ] ORCID IDs included (if required)
- [ ] All authors have approved submission
- [ ] Author contributions documented

### Abstract

- [ ] Contains all required elements:
  - [ ] Context/motivation
  - [ ] Problem/gap
  - [ ] Approach/methods
  - [ ] Key results (quantified)
  - [ ] Significance/implications
- [ ] Within word limit: ___/___ words
- [ ] No citations
- [ ] No undefined acronyms
- [ ] Matches manuscript content
- [ ] Standalone (understandable without paper)

### Keywords

- [ ] Keywords selected (typically 3-6)
- [ ] Terms appear in title/abstract
- [ ] Include searchable terms
- [ ] Keywords: _______________________

### Introduction

- [ ] Opens with broad context
- [ ] Provides sufficient background
- [ ] Fairly represents prior work
- [ ] Clearly states gap/problem
- [ ] Explicitly states contribution
- [ ] Novelty is evident
- [ ] Appropriate length for journal

### Methods/Theory

- [ ] Sufficient detail for reproduction
- [ ] Approach clearly justified
- [ ] Assumptions stated
- [ ] Controls described (if applicable)
- [ ] Statistical methods specified
- [ ] Equipment/software identified
- [ ] References to established methods

### Results

- [ ] Logically organized
- [ ] Each result clearly stated
- [ ] Quantitative with uncertainties
- [ ] All figures/tables referenced
- [ ] Data interpretation in Discussion
- [ ] No unexplained anomalies

### Discussion

- [ ] Summarizes key findings
- [ ] Interprets results in context
- [ ] Compares with prior work
- [ ] Addresses alternative explanations
- [ ] Acknowledges limitations
- [ ] Discusses implications
- [ ] Suggests future directions

### Conclusions

- [ ] Concise summary of contribution
- [ ] Main findings restated
- [ ] Distinct from abstract
- [ ] Forward-looking element
- [ ] No new information

### Acknowledgments

- [ ] Funding sources listed with grant numbers
- [ ] Technical assistance acknowledged
- [ ] Facility access acknowledged
- [ ] Helpful discussions acknowledged (if appropriate)

## Part IV: Writing Quality

### Clarity

- [ ] No ambiguous sentences
- [ ] Technical terms defined
- [ ] Acronyms defined on first use
- [ ] No unnecessarily complex sentences
- [ ] Consistent terminology throughout

### Precision

- [ ] All numerical values verified
- [ ] Claims match evidence level
- [ ] Uncertainty stated for all measurements
- [ ] No vague quantifiers ("several," "many")
- [ ] Notation consistent throughout

### Concision

- [ ] No unnecessary words/phrases
- [ ] No redundant paragraphs
- [ ] Within journal length limits
- [ ] Word count: ___/___ (limit)

### Flow

- [ ] Clear topic sentences
- [ ] Smooth transitions between paragraphs
- [ ] Logical section progression
- [ ] Reader can follow narrative

### Style

- [ ] Appropriate use of active/passive voice
- [ ] Varied sentence structure
- [ ] Professional tone
- [ ] No colloquialisms

### Grammar and Mechanics

- [ ] Spell-check completed
- [ ] Grammar-check completed
- [ ] Subject-verb agreement verified
- [ ] Tense consistency verified
- [ ] No typos

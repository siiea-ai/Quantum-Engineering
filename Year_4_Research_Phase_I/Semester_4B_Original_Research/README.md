## Paper Writing Framework

### Target Journal Options

| Journal | Focus | Length | IF |
|---------|-------|--------|-----|
| Physical Review A | QI, atomic physics | 6-15 pages | 2.9 |
| Physical Review X | High-impact | 10-20 pages | 12.6 |
| Quantum | QI focus | 10-20 pages | 6.0 |
| npj Quantum Information | Broad QI | 8-15 pages | 5.8 |
| arXiv (preprint) | All | Any | N/A |

### Paper Section Checklist

| Section | Elements | Done |
|---------|----------|------|
| **Title** | Concise, descriptive, <15 words | ☐ |
| **Abstract** | Problem, method, results, impact (150-250 words) | ☐ |
| **Introduction** | Context, gap, contribution, outline | ☐ |
| **Background** | Prior work, theoretical framework | ☐ |
| **Methods** | Reproducible methodology | ☐ |
| **Results** | Clear presentation, figures, analysis | ☐ |
| **Discussion** | Interpretation, limitations, implications | ☐ |
| **Conclusion** | Summary, future work | ☐ |
| **References** | Complete, properly formatted | ☐ |
| **Supplementary** | Additional data, proofs, code | ☐ |

## Code Repository Standards

### Repository Structure

```
project-name/
├── README.md                 # Project overview, installation, usage
├── LICENSE                   # MIT or Apache 2.0
├── requirements.txt          # Python dependencies
├── setup.py                  # Package installation
├── .gitignore               # Standard Python ignores
├── src/
│   ├── __init__.py
│   ├── core/                # Core algorithms
│   ├── utils/               # Utility functions
│   └── experiments/         # Experiment scripts
├── tests/
│   └── test_*.py            # Unit tests
├── notebooks/
│   └── *.ipynb              # Jupyter notebooks
├── data/
│   ├── raw/                 # Raw data (if shareable)
│   └── processed/           # Processed data
├── figures/
│   └── *.pdf                # Publication figures
└── docs/
    └── *.md                 # Documentation
```

## Assessment Criteria

### Research Quality

| Criterion | Description | Weight |
|-----------|-------------|--------|
| **Novelty** | Original contribution to knowledge | 25% |
| **Rigor** | Methodological soundness | 25% |
| **Significance** | Impact on field | 20% |
| **Clarity** | Communication quality | 15% |
| **Reproducibility** | Code and data availability | 15% |

### Paper Quality Rubric

| Aspect | Excellent | Good | Needs Work |
|--------|-----------|------|------------|
| **Abstract** | Clear, complete, compelling | Missing element | Unclear |
| **Introduction** | Strong motivation, clear gap | Adequate context | Weak motivation |
| **Methods** | Fully reproducible | Minor gaps | Not reproducible |
| **Results** | Clear, well-presented | Some confusion | Poorly organized |
| **Figures** | Publication quality | Minor issues | Unprofessional |
| **Writing** | Polished, clear | Some awkwardness | Significant issues |

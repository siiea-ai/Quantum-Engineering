# First Draft Completion Checklist

## Overview

This checklist ensures your first draft is complete and ready for revision. Complete each section before considering your draft finished. The goal is completeness, not perfectionâ€”revision will address quality issues.

---

## Section Completion

### Title
- [ ] Working title exists
- [ ] Title is specific and descriptive
- [ ] Contains searchable keywords
- [ ] Appropriate length (10-15 words typical)

### Authors and Affiliations
- [ ] All authors listed
- [ ] Author order determined
- [ ] Affiliations complete and accurate
- [ ] Corresponding author identified

### Abstract
- [ ] Context established (1-2 sentences)
- [ ] Problem/gap stated (1 sentence)
- [ ] Approach indicated (1 sentence)
- [ ] Key results with numbers (2-3 sentences)
- [ ] Implications stated (1 sentence)
- [ ] Within word limit
- [ ] No references or undefined abbreviations

### Introduction
- [ ] Hook engages reader (Paragraph 1)
- [ ] Background establishes context (Paragraphs 2-3)
- [ ] Gap clearly stated (Paragraph 4)
- [ ] Contribution specified (Paragraph 5)
- [ ] Paper organization (optional, Paragraph 6)
- [ ] All key references included
- [ ] Flows logically from broad to specific

### Methods/Theory
- [ ] System/model described
- [ ] Key parameters specified
- [ ] Procedures detailed for reproducibility
- [ ] Established methods referenced
- [ ] Data analysis methods explained
- [ ] Approximations acknowledged
- [ ] Equations numbered

### Results
- [ ] All major findings presented
- [ ] Each claim supported by data
- [ ] Figures referenced in order
- [ ] Quantitative values with uncertainties
- [ ] Objective presentation (no interpretation)
- [ ] Negative results included if relevant
- [ ] Logical organization

### Discussion
- [ ] Main findings summarized
- [ ] Results interpreted
- [ ] Comparison with prior work
- [ ] Limitations acknowledged
- [ ] Implications discussed
- [ ] Future directions suggested
- [ ] Distinct from Results

### Conclusions
- [ ] Main achievement stated
- [ ] Key results summarized
- [ ] Significance indicated
- [ ] Future outlook provided
- [ ] Distinct from Abstract

### Acknowledgments
- [ ] Funding sources listed (with grant numbers)
- [ ] Facilities acknowledged
- [ ] Helpful discussions acknowledged
- [ ] Technical assistance credited

### References
- [ ] All citations have entries
- [ ] No unused references
- [ ] Format consistent with journal style
- [ ] Each reference verified for accuracy
- [ ] Preprints updated to published versions
- [ ] Numbering correct (no gaps)

### Supplementary Material (if applicable)
- [ ] All SM sections complete
- [ ] Main text references SM appropriately
- [ ] SM figures/tables numbered correctly
- [ ] SM formatted according to guidelines

---

## Figure Completion

### Figure Quality
- [ ] All figures present
- [ ] Resolution adequate (300+ DPI)
- [ ] Readable at final print size
- [ ] Consistent style across figures
- [ ] Colorblind-friendly

### Figure Captions
- [ ] Each figure has complete caption
- [ ] Captions are self-contained
- [ ] All symbols/colors defined
- [ ] Error bar meaning specified
- [ ] Key parameters included

### Figure Integration
- [ ] All figures referenced in text
- [ ] Figure numbering sequential
- [ ] References match figure numbers
- [ ] Placement appropriate

---

## Equation Completion

- [ ] All equations present
- [ ] Key equations numbered
- [ ] Variables defined when introduced
- [ ] Notation consistent throughout
- [ ] Units included where appropriate
- [ ] Equation references correct

---

## Internal Consistency

### Cross-References
- [ ] Figure references correct (Fig. 1, not Fig. 2)
- [ ] Equation references correct
- [ ] Section references correct (if used)
- [ ] Table references correct

### Parameter Consistency
- [ ] Parameter values consistent throughout
- [ ] Notation consistent between sections
- [ ] Figures use same values as text

### Terminology Consistency
- [ ] Terms used consistently
- [ ] Abbreviations defined on first use
- [ ] Consistent capitalization

---

## Formatting

### Document Format
- [ ] Journal template applied (if available)
- [ ] Fonts consistent
- [ ] Margins correct
- [ ] Line spacing as required
- [ ] Page numbers present (if required)

### Length Check
- [ ] Within journal page/word limits
- [ ] Identify sections to cut if over limit
- [ ] Main text vs. supplementary balance appropriate

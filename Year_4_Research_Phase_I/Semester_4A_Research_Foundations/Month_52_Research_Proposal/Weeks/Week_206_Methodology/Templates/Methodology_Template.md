## Methodology Checklist

Before finalizing, verify:

### Completeness
- [ ] All aims have detailed methodology
- [ ] All methods have specific parameters
- [ ] All methods have validation approaches
- [ ] All major risks are addressed

### Specificity
- [ ] Tools and software are named with versions
- [ ] Numerical parameters are provided
- [ ] Sample sizes/replicates are specified
- [ ] Success criteria are quantitative

### Feasibility
- [ ] Methods match available resources
- [ ] Timeline is realistic with buffer
- [ ] Personnel effort is appropriate
- [ ] Computing needs are justified

### Rigor
- [ ] Controls are specified
- [ ] Statistical analysis is described
- [ ] Validation against known results is planned
- [ ] Alternative approaches are provided

### Integration
- [ ] Aims connect logically
- [ ] Dependencies are clear
- [ ] Critical path is identified
- [ ] Parallel activities are noted

# Week 212: Scientific Writing Tools

## Days 1478-1484 | Year 4, Semester 4A

## Overview

This week covers professional scientific writing tools essential for publishing quantum computing research. You will master LaTeX, learn REVTeX for physics journals, explore Overleaf for collaborative writing, and develop skills in figure preparation and bibliography management.

**Prerequisites:** Basic document preparation, prior research experience

**Learning Outcomes:**
- Write professional physics papers using LaTeX and REVTeX
- Use Overleaf for collaborative manuscript development
- Create publication-quality figures with proper formatting
- Manage bibliographies with BibTeX and reference managers
- Understand journal submission requirements
- Develop effective scientific writing practices

## Writing Checklist

### Before Submission

- [ ] Abstract clearly states contribution
- [ ] All figures are high resolution (300+ dpi)
- [ ] References are complete and consistent
- [ ] Equations are numbered appropriately
- [ ] Acknowledgments include funding
- [ ] Supplementary materials are organized
- [ ] Author contributions are specified
- [ ] Conflict of interest statement included

### Style Guidelines

- [ ] Consistent notation throughout
- [ ] Defined all symbols on first use
- [ ] Active voice preferred
- [ ] One idea per paragraph
- [ ] Clear topic sentences
- [ ] Logical flow between sections

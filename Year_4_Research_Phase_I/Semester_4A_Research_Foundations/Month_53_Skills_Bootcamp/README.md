## Common Pitfalls to Avoid

1. **Premature Optimization:** Profile before optimizing; focus on algorithmic improvements first
2. **Framework Lock-in:** Learn multiple tools; each has strengths
3. **Neglecting Testing:** Bugs in research code invalidate results
4. **Poor Documentation:** Future-you will thank present-you
5. **Ignoring Version Control:** Commits are checkpoints for your research
6. **Copy-Paste Programming:** Understand code before using it

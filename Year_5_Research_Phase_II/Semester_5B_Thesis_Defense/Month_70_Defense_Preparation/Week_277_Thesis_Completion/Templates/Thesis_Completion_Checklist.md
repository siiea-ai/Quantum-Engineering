## Part 1: Front Matter

### Title Page
- [ ] University name correct and properly styled
- [ ] Full thesis title (check for typos)
- [ ] Subtitle (if applicable) properly formatted

- [ ] Department name correct
- [ ] Degree name correct (Doctor of Philosophy)
- [ ] Field/subject area correct
- [ ] Month and year of degree conferral
- [ ] Formatting matches institutional template

### Copyright Page
- [ ] Copyright year correct
- [ ] Author name matches title page
- [ ] Rights statement appropriate

### Abstract
- [ ] Word count within limit (typically 350 words): _____ words
- [ ] Research problem clearly stated
- [ ] Methods briefly described
- [ ] Key findings summarized (all major contributions)
- [ ] Significance/implications stated
- [ ] No citations (typically not allowed)
- [ ] No undefined abbreviations
- [ ] Standalone readability verified

### Acknowledgments
- [ ] Thesis advisor acknowledged
- [ ] Committee members acknowledged
- [ ] Research collaborators acknowledged
- [ ] Lab members and peers acknowledged
- [ ] Funding sources listed (grant numbers if required)
- [ ] Family/personal acknowledgments included
- [ ] No one important omitted

### Dedication (if included)
- [ ] Appropriate tone and length
- [ ] Properly formatted

### Table of Contents
- [ ] All chapters listed
- [ ] All sections listed to appropriate depth
- [ ] Page numbers correct
- [ ] Heading text matches actual headings
- [ ] Consistent formatting throughout

### List of Figures
- [ ] All figures listed
- [ ] Captions match actual figure captions
- [ ] Page numbers correct
- [ ] Figure numbers sequential

### List of Tables
- [ ] All tables listed
- [ ] Captions match actual table captions
- [ ] Page numbers correct
- [ ] Table numbers sequential

### List of Abbreviations/Nomenclature (if included)
- [ ] All abbreviations defined
- [ ] Alphabetically ordered
- [ ] Consistent with usage in text

## Part 5: Formatting

### Document Specifications
- [ ] Correct margins: ___ inches (typically 1" or 1.5")
- [ ] Correct font: ____________ (typically Times or Computer Modern)
- [ ] Correct font size: ___ pt (typically 11 or 12 pt)
- [ ] Correct line spacing: ____________ (typically double)
- [ ] Paragraph spacing appropriate

### Page Numbering
- [ ] Front matter uses roman numerals (i, ii, iii...)
- [ ] Body uses arabic numerals (1, 2, 3...)

- [ ] Title page is page i (but number not shown)

### Headers and Footers
- [ ] Format matches institutional requirements
- [ ] Chapter titles in headers (if required)
- [ ] Page numbers consistent

### Section Formatting
- [ ] Chapter titles formatted correctly
- [ ] Section headings hierarchical (1.1, 1.1.1)
- [ ] Heading levels visually distinct
- [ ] Consistent capitalization in headings

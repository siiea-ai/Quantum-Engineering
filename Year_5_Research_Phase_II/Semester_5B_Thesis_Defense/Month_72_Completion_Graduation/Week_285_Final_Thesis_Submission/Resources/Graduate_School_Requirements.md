# Graduate School Submission Requirements

## Overview

This document outlines typical requirements for doctoral thesis submission. Requirements vary by institution---always consult your graduate school's official guidelines.

---

## Timeline

### Typical Submission Timeline

| Milestone | Timing Before Conferral |
|-----------|------------------------|
| Defense | 8-12 weeks |
| Final revisions | 6-8 weeks |
| Committee approval | 4-6 weeks |
| Format check | 3-4 weeks |
| Final submission | 2-3 weeks |
| Degree conferral | Graduation date |

### Key Deadlines

Most universities have firm deadlines tied to graduation dates:

**Fall Graduation:** October-November submission
**Spring Graduation:** March-April submission
**Summer Graduation:** June-July submission

**Important:** Missing the deadline typically delays graduation by one semester.

## Required Documents

### 1. Final Thesis Document

**Format Requirements:**
- PDF format (PDF/A preferred)
- All fonts embedded
- No password protection
- File size typically under 100MB
- Accessible (tagged PDF preferred)

**Filename Convention:**
```
LastName_FirstName_PhD_Year.pdf
Example: Smith_Jane_PhD_2026.pdf
```

### 2. Approval/Signature Page

**Required Signatures:**
- Dissertation advisor
- All committee members
- Department chair (some institutions)
- Dean's representative (some institutions)

**Signature Options:**
- Physical signatures (scanned)
- Digital signatures (DocuSign, institutional system)
- Electronic approval through university system

### 3. Survey of Earned Doctorates (SED)

**Federal Requirement:**
- Required by National Science Foundation
- Collects data on doctoral degree recipients
- Must complete before submission accepted
- Typically online at [sed-ncses.org](https://sed-ncses.org)

### 4. Publishing Agreement

**ProQuest/UMI Agreement:**
- Traditional publishing (database inclusion)
- Open access option (additional fee)
- Bound copy orders

**Institutional Repository Agreement:**
- License grant to university
- Access level selection
- Preservation terms

### 5. Abstract

**Requirements:**
- Typically 150-350 word limit
- Plain text (no special characters)
- Summarizes research and findings
- Keywords (5-10 terms)

## Publishing Options

### Traditional Publishing

**ProQuest Dissertations & Theses:**
- Included in ProQuest database
- Available for purchase
- Indexed and searchable
- No cost to student

### Open Access Publishing

**Benefits:**
- Freely accessible worldwide
- Higher citation rates
- Broader impact
- Immediate availability

**Costs:**
- ProQuest Open Access: ~$95
- Institutional repository: Usually free
- Some institutions subsidize fees

### Embargo/Restriction Options

**Reasons for Embargo:**
- Pending patent applications
- Forthcoming publications
- Sensitive data
- Contractual obligations

**Typical Options:**
- 6 months
- 1 year
- 2 years
- Longer (requires justification)

**Request Process:**
- Written justification required
- Advisor approval
- Graduate dean approval
- Renewable if needed

## Copyright Considerations

### Copyright Registration

**Optional but recommended:**
- Establishes legal record
- Required for statutory damages
- Cost: ~$65 (electronic filing)
- Through [copyright.gov](https://www.copyright.gov)

**Note:** Copyright exists automatically upon creation. Registration is optional.

### Thesis Copyright

**Standard approach:**
- Student retains copyright
- Grant license to university for distribution
- Can publish parts elsewhere

**Third-party materials:**
- Must have permission for copyrighted figures/content
- Document permissions in thesis
- Fair use applies to limited excerpts

## Technical Requirements

### PDF Specifications

**Required:**
- PDF version 1.4 or higher
- All fonts embedded
- No security/password protection
- 300+ DPI images

**PDF/A (Archival) Format:**
- Required by some institutions
- Ensures long-term accessibility
- Specific font requirements
- No external dependencies

**Creating PDF/A:**
```latex
% LaTeX
\usepackage[a-1b]{pdfx}
```

### File Size Management

**If file is too large:**
1. Compress images using optimization tools
2. Reduce image resolution (maintain 300 DPI minimum)
3. Use vector graphics where possible
4. Submit supplementary materials separately

### Accessibility Requirements

**Increasingly required:**
- Document language specified
- Heading structure (bookmarks)
- Alt text for figures
- Proper reading order
- Tagged PDF

## Format Check Process

### Initial Review

Graduate school staff review for:
- Margin compliance
- Page number accuracy
- Font consistency
- Required sections present
- Signature page complete

### Common Issues

| Issue | Solution |
|-------|----------|
| Margin violations | Adjust in document template |
| Inconsistent fonts | Ensure all fonts embedded |
| Missing page numbers | Check LaTeX configuration |
| ToC errors | Regenerate after final edits |
| Low-resolution figures | Replace with 300+ DPI versions |

### Revision Cycle

1. Submit for format check
2. Receive feedback (usually 3-5 business days)
3. Make corrections
4. Resubmit
5. Repeat until approved
6. Final approval received

**Tip:** Many institutions offer preliminary format checks before defense.

## Post-Submission

### What Happens After Submission

1. **Format verification** - Final check by graduate school
2. **Database entry** - Added to university and ProQuest systems
3. **Binding** - If personal copies ordered
4. **Indexing** - Available in dissertation databases
5. **Degree conferral** - Official graduation date

### Confirmation Documents

**You should receive:**
- Submission confirmation email
- Confirmation number/ID
- Expected processing timeline
- Notification of any issues

### Degree Conferral

**Timing:**
- Degrees typically conferred on specific dates (3x per year)
- Diplomas mailed after conferral
- Transcript updated after conferral

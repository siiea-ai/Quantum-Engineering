## Formatting Compliance

### Page Layout
- [ ] Paper size: _____ (typically 8.5" x 11" or A4)
- [ ] Top margin: _____ inch
- [ ] Bottom margin: _____ inch
- [ ] Left margin: _____ inch (binding side)
- [ ] Right margin: _____ inch
- [ ] Consistent margins throughout

### Typography
- [ ] Font: _____ (typically Times New Roman or similar)
- [ ] Font size: 12pt for body text
- [ ] Line spacing: Double-spaced (body text)
- [ ] Line spacing: Single-spaced (block quotes, captions, footnotes, bibliography)
- [ ] Paragraph indentation consistent

### Page Numbers
- [ ] Preliminary pages: lowercase Roman numerals (ii, iii, iv...)
- [ ] Title page: no number (counts as i)
- [ ] Body: Arabic numerals starting at 1
- [ ] Position: _____ (typically centered bottom or upper right)
- [ ] No page number on chapter opening pages (if required)

### Headings
- [ ] Chapter titles: centered, bold, consistent format
- [ ] Section headings: consistent hierarchy
- [ ] Subsection headings: consistent hierarchy
- [ ] No orphaned headings (heading at bottom of page without content)

### Figures and Tables
- [ ] Figure captions below figure
- [ ] Table captions above table
- [ ] Caption format consistent throughout
- [ ] All figures minimum 300 DPI resolution
- [ ] All text in figures readable at print size
- [ ] Permission statements for copyrighted material included

### Citations and Bibliography
- [ ] Citation style: _____ (APA, IEEE, Physical Review, etc.)
- [ ] All citations present in bibliography
- [ ] No unused bibliography entries
- [ ] Bibliography format consistent
- [ ] DOIs included where available

## Night Before (Complete by 10 PM)

### Materials Preparation
- [ ] Laptop fully charged
- [ ] Laptop charger packed
- [ ] USB drive with presentation
- [ ] Laser pointer with fresh batteries
- [ ] Backup batteries for pointer
- [ ] Printed presenter notes
- [ ] Printed thesis copy (for reference)
- [ ] Watch or timer
- [ ] Water bottle (filled)
- [ ] Light snacks (energy bar, nuts)
- [ ] Tissues
- [ ] Any personal comfort items

### Clothes and Personal Items
- [ ] Professional attire selected and laid out
- [ ] Comfortable shoes ready
- [ ] Backup option for spills/weather
- [ ] Personal hygiene items

### Logistics
- [ ] Transportation plan confirmed
- [ ] Route checked (traffic, parking, transit)
- [ ] Arrival time calculated (30+ min early)
- [ ] Building access confirmed
- [ ] Room location known

### Mental Preparation
- [ ] Relaxation exercises completed
- [ ] Opening statement reviewed briefly
- [ ] Positive visualization done
- [ ] Caffeine intake limited after 2 PM
- [ ] Light dinner eaten (nothing heavy or unusual)
- [ ] Alcohol avoided

### Sleep Preparation
- [ ] Alarms set (at least 2)
- [ ] Phone charged
- [ ] Sleep environment prepared
- [ ] Bedtime by 10 PM

## Defense Morning

### Upon Waking (T-3 hours)
- [ ] Adequate sleep achieved (7-8 hours)
- [ ] Feeling reasonably rested
- [ ] Alarms worked correctly
- [ ] No illness symptoms

### Morning Routine (T-3 to T-2 hours)
- [ ] Shower completed
- [ ] Professional attire put on
- [ ] Light, balanced breakfast eaten
  - Recommended: protein, complex carbs, fruit
  - Avoid: heavy/greasy foods, too much caffeine, unfamiliar foods
- [ ] Water consumed (hydration important)
- [ ] Presentation briefly reviewed (15 min max)
  - Do NOT over-reviewâ€”you're ready
- [ ] Deep breathing exercises (5 minutes)

### Final Preparation (T-2 to T-1.5 hours)
- [ ] All materials double-checked
- [ ] Bag packed with everything
- [ ] Phone charged and silenced
- [ ] Emergency contacts accessible
- [ ] Keys and wallet/ID ready
- [ ] Weather-appropriate items (umbrella, coat)

### Departure (T-1.5 hours)
- [ ] Left with adequate buffer time
- [ ] Transportation going as planned
- [ ] Arrival estimate still on track

## At the Venue

### Arrival (T-1 hour)
- [ ] Arrived at building
- [ ] Building access worked
- [ ] Room located
- [ ] Restroom visited
- [ ] Water/snacks accessible

### Room Setup (T-45 to T-30 minutes)

**Technology Setup:**
- [ ] Room lights work
- [ ] Projector powered on
- [ ] Laptop connected successfully
- [ ] Presentation displaying correctly
- [ ] Resolution/aspect ratio correct
- [ ] Colors displaying properly
- [ ] Animations working
- [ ] Presenter notes visible (if using)
- [ ] Laser pointer tested
- [ ] Slide advancement working
- [ ] Audio tested (if needed)

**Remote Setup (if applicable):**
- [ ] Video conference connected
- [ ] Camera positioned correctly
- [ ] Microphone working
- [ ] Remote attendees can see slides
- [ ] Remote attendees can hear clearly
- [ ] You can hear remote attendees
- [ ] Recording started (if required)

**Physical Setup:**
- [ ] Podium/table positioned
- [ ] Water accessible at speaking position
- [ ] Clear path for moving if needed
- [ ] Timer visible
- [ ] Whiteboard available and has markers
- [ ] Chair arrangement appropriate

### Pre-Defense (T-30 to T-15 minutes)
- [ ] One final restroom visit
- [ ] Water bottle refilled
- [ ] Deep breathing (5-10 breaths)
- [ ] Positive self-talk completed
- [ ] Opening statement mentally reviewed
- [ ] Smile practiced

### Committee Arrival (T-15 minutes to start)
- [ ] Greet committee members warmly
- [ ] Thank them for their time
- [ ] Brief casual conversation (weather, travel)
- [ ] Remain calm and professional
- [ ] Phone completely silenced

## During Defense

### Opening (First 5 minutes)
- [ ] Deep breath before starting
- [ ] Thank committee and audience
- [ ] Introduce yourself and thesis title
- [ ] Provide brief roadmap
- [ ] Begin with confidence

### Presentation Monitoring
- [ ] Watching time periodically
- [ ] Adjusting pace as needed
- [ ] Making eye contact with committee
- [ ] Speaking clearly and loudly enough
- [ ] Using pointer effectively
- [ ] Staying calm if interruptions occur
- [ ] Taking water breaks if needed

### Presentation Completion
- [ ] Finished within time limit (45-60 min)
- [ ] Covered all key points
- [ ] Ended with clear conclusions
- [ ] Thanked committee for attention
- [ ] Indicated readiness for questions

---

## Q&A Session

### Throughout Q&A
- [ ] Listening carefully to each question
- [ ] Pausing to think before answering
- [ ] Asking for clarification if needed
- [ ] Answering directly and concisely
- [ ] Acknowledging limitations honestly
- [ ] Referencing backup slides when helpful
- [ ] Maintaining composure under challenge
- [ ] Taking notes on important feedback
- [ ] Thanking questioners

### After Each Question
- [ ] Confirmed questioner is satisfied
- [ ] Ready for next question
- [ ] Remaining calm and focused

---

## Post-Defense

### Immediately After
- [ ] Committee departs for deliberation (if applicable)
- [ ] Wait calmly for result
- [ ] Stay in or near the room

### Receiving Results
- [ ] Result received: ____________________
  - [ ] Pass (unconditional)
  - [ ] Pass with minor revisions
  - [ ] Pass with major revisions
  - [ ] Deferred (re-defense required)
- [ ] Revision requirements documented
- [ ] Timeline for revisions confirmed
- [ ] Next steps clarified

### Post-Result Actions
- [ ] Thanked committee individually
- [ ] Obtained any written feedback
- [ ] Confirmed revision submission process
- [ ] Gathered all personal items
- [ ] Room left clean

### Celebration/Decompression
- [ ] Contacted family/friends with news
- [ ] Celebrated appropriately
- [ ] Gave yourself credit for completing
- [ ] Rested and recovered

## If Issues Arise

### Technical Problems During Defense

| Problem | Solution | Completed |
|---------|----------|-----------|
| Projector fails | Switch to PDF on backup computer | [ ] |
| Laptop crashes | Use USB on venue computer | [ ] |
| Remote connection fails | Use phone backup dial-in | [ ] |
| Slides not advancing | Use keyboard arrows directly | [ ] |
| Pointer fails | Use mouse cursor or verbal direction | [ ] |

### Personal Challenges

| Challenge | Response |
|-----------|----------|
| Anxiety spike | Deep breaths, ground yourself, take water |
| Mind goes blank | Pause, breathe, check slides, ask to revisit |
| Don't know answer | "That's an excellent point I'll need to explore further" |
| Hostile question | Stay calm, address substance, don't become defensive |
| Fatigue | Request brief pause if needed |

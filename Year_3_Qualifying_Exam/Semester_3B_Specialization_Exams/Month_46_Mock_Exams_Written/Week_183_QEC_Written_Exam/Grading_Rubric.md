## QEC-Specific Common Errors

### Stabilizer Formalism
- Incorrect commutation counting
- Confusing stabilizers with logical operators
- Wrong weight calculation

### CSS Codes
- Mixing $X$ and $Z$ stabilizer roles
- Incorrect Hamming code structure
- Missing transversal gate analysis

### Surface Codes
- Wrong boundary conditions
- Incorrect syndrome pattern
- Distance miscalculation

### Fault Tolerance
- Imprecise definition
- Missing Eastin-Knill implications
- Wrong overhead estimates

### Threshold
- Incorrect exponent in concatenation
- Missing assumptions
- Unrealistic resource estimates

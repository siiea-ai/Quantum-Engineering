## Key Equations to Master

### Quantum Mechanics
- Schrödinger equation: $$i\hbar\frac{\partial}{\partial t}|\psi\rangle = H|\psi\rangle$$
- Uncertainty principle: $$\Delta A \Delta B \geq \frac{1}{2}|\langle[A,B]\rangle|$$
- Angular momentum: $$[L_i, L_j] = i\hbar\epsilon_{ijk}L_k$$
- Perturbation theory: $$E_n^{(1)} = \langle n^{(0)}|H'|n^{(0)}\rangle$$

### Quantum Information
- Von Neumann entropy: $$S(\rho) = -\text{Tr}(\rho \log \rho)$$
- Fidelity: $$F(\rho, \sigma) = \left(\text{Tr}\sqrt{\sqrt{\rho}\sigma\sqrt{\rho}}\right)^2$$
- Quantum channel: $$\mathcal{E}(\rho) = \sum_k K_k \rho K_k^\dagger$$
